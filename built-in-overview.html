Built-in Components Overview

Model NCPipeline contains chain of components traits which are responsible for sentence processing. Some built-in implementations of these traits are described below.

There are two kinds of pipeline components:

  • Following pipeline components have built-in implementations and described in related sections: NCTokenParser, NCTokenEnricher, NCEntityParser.
  • Following pipeline components cannot have build implementation because their logic are depended on concrete user model: NCTokenValidator, NCEntityEnricher, NCEntityValidator, NCEntityMapper and NCVariantFilter.
