Overview

bin/nlpcraft.{sh|cmd} script that is a central management utility for NLPCraft. Most of the functions this script does can be done through running Java applications - yet, using this script, you can accomplish the same things much simpler and quicker. From starting and managing REST server, running various tools, to testing models via REST API - bin/nlpcraft.{sh|cmd} script provides a single centralized utility that makes a quick work out of all these routine tasks.

The script is only available in either official Apache source ZIP or in binary ZIP distributions. If you used Maven or similar build tools to add NLPCraft to your project, this script will not available automatically - you will need to download ZIP distribution separately.

ZIP Distribution

The bin/nlpcraft.{sh|cmd} script is only available in ZIP distributions. For Maven/Gradle/SBT projects you will need to download ZIP distribution separately.

The script is located in /bin sub-folder of the installation root and is available for Unix/Linux (nlpcraft.sh) and Windows (nlpcraft.cmd) environments. We will use .sh version for the rest of instructions unless specified otherwise.
