Built-in Token Enricher

NCTokenEnricher is a component which allows to add additional properties to prepared tokens, like part of speech, quote, stop-words flags or any other. NLPCraft provides English language default set of token enrichers implementations.

Lemma And POS Enricher

NCOpenNLPLemmaPosTokenEnricher - this component allows to add lemma and pos values to processed token. Look at these links fpr more details: Lemma and Part of speech. Current implementation is based on Apache OpenNLP project components. Is uses Apache OpenNLP models, which are accessible here for POS taggers. English lemmatization model is accessible here. You can use any models which are compatible with Apache OpenNLP POSTaggerME and DictionaryLemmatizer components.

Brackets Enricher

NCEnBracketsTokenEnricher - this component allows to add brackets boolean flag to processed token.

Quotes Enricher

NCEnQuotesTokenEnricher - this component allows to add quoted boolean flag to processed token.

Dictionary Enricher

NCEnDictionaryTokenEnricher - this component allows to add dict boolean flag to processed token. Note that it requires already defined lemma token property. You can use NCOpenNLPLemmaPosTokenEnricher or any another component which sets lemma into the token. Note that you have to define it in model pipilene token enricher list before NCEnDictionaryTokenEnricher.

Stop-words Enricher

NCEnStopWordsTokenEnricher - this component allows to add stopword boolean flag to processed token. It is based on predefined rules for English language, but it can be also extended by custom user word list and excluded list. Note that it requires already defined lemma token property. You can use NCOpenNLPLemmaPosTokenEnricher or any another component which sets lemma into the toke. Note that you have to define it in model pipilene token enricher list before NCEnStopWordsTokenEnricher.

Swear-words Enricher

NCEnSwearWordsTokenEnricher - this component allows to add swear boolean flag to processed token.
